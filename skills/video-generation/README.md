# Video Generation Skill: Camera Movements

This skill empowers Antigravity to simulate and implement professional camera movements on video content. It covers everything from basic Pans and Tilts to complex cinematic effects like the Dolly Zoom (Vertigo Effect).

## Quick Start
1. **Try the Demo**: Open `examples/web-app/index.html` in your browser to see a live simulation of all 13 movements.
2. **Implementation**: Refer to `SKILL.md` for CSS and Web Animation API implementation details.
3. **Advanced**: Check `resources/ffmpeg-guide.md` for programmatic video processing commands.

## Features
- **13 Cinematic Movements**: Detailed descriptions and logic for all standard camera shots.
- **Premium UI Example**: A glassmorphic web app for real-time video manipulation.
- **FFMPEG Integration**: Syntax for server-side video generation.

## Directory Structure
- `SKILL.md`: Core logic and AI instructions.
- `examples/web-app/`: A fully functional demonstration tool.
- `resources/`: Technical guides and cheatsheets.
