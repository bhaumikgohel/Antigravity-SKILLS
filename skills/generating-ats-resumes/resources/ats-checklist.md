Verify the generated or reviewed resume against these points:

| Checkpoint | Requirement | Status |
| :--- | :--- | :---: |
| **Keyword Match** | Does the resume use the exact wording from the Job Description? | [ ] |
| **Job Titles** | Do the recent job titles align with the target role? | [ ] |
| **Formatting** | Is the layout single-column with no tables or graphics? | [ ] |
| **Hard Skills** | Are technical skills/tools listed prominently in the Skills section? | [ ] |
| **Metrics** | Are achievements quantified with numbers/percentages? | [ ] |
| **File Safety** | Is it suitable for .docx or clean PDF export (no headers/footers)? | [ ] |
| **Action Verbs** | Does each bullet point start with a strong, active verb? | [ ] |
| **Reverse Chrono** | Is the work experience in reverse chronological order? | [ ] |

## ATS Ranking Heuristics
- **High Rank**: Keywords match JD precisely + titles align + metrics included.
- **Medium Rank**: Keywords present but titles differ or metrics missing.
- **Auto-Rejection Risk**: Uses tables, columns, or non-standard font icons.
